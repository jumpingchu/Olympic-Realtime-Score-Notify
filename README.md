# Olympic Realtime Score Notify (奧運即時比分 LINE 通知)

## Overview

* Get live/realtime score from Google search results and receive notification by LINE Notify.

* 從 Google 搜尋奧運比分的結果中，擷取即時比分資訊，並使用 LINE Notify 接收比數通知
